The version system I'm using might be confusing, so here is an explanation:

Note: The Client and APWorld will use one version number. An update may effect only the Client, the APWorld, or both. This will be specified in the release notes.

A version number will be in the format of a.b.c
- a
  - These are major updates
  - Most likely if multiple minor updates dealing with the same thing will be turned into one major update.
- b
  - These are the minor updates
  - Maybe bug fixes, some items are added, locations added, etc.
- c
  - These are minor updates
  - The tiny things like bug fixes.
